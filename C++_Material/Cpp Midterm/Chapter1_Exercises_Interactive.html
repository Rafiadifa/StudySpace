Exercise 1.1 Basic

Fix the Errors

Identify and fix the 5 errors in this code. Look for type mismatches, incorrect quotes, and uninitialized variables.

#include <iostream> #include <string> using namespace std; int main() { int age = 18.5; // Error 1 string name = 'Alice'; // Error 2 char grade = "A"; // Error 3 double price = 10 / 3; // Error 4 bool isStudent; // Error 5 cout << "Age: " << age << endl; cout << "Name: " << name << endl; cout << "Grade: " << grade << endl; cout << "Price: " << price << endl; cout << "Is student: " << isStudent << endl; return 0; }
Hint: Watch out for: type mismatches (int vs double), quotes (single vs double), division types, and variable initialization!
Solution:
  1. int age = 18.5 - Assigning double to int truncates to 18. Should be double age = 18.5
  2. string name = 'Alice' - Strings need double quotes. Should be string name = "Alice"
  3. char grade = "A" - Chars need single quotes. Should be char grade = 'A'
  4. double price = 10 / 3 - Integer division gives 3, not 3.33. Should be double price = 10.0 / 3
  5. bool isStudent - Uninitialized variable. Should be bool isStudent = true
Exercise 1.2 Basic

Variable Swap

Swap the values of two variables a = 5 and b = 10 without directly assigning new values. You must use a temporary variable.

int a = 5; int b = 10; cout << "Before: a = " << a << ", b = " << b << endl; // TODO: Write your swap logic here cout << "After: a = " << a << ", b = " << b << endl;
Hint: If you do a = b first, you lose the original value of a. Create a temporary variable to save it first!
Solution:
int temp = a; // temp now holds 5 a = b; // a now holds 10 b = temp; // b now holds 5

Key Insight: Variable assignment is destructive - it overwrites the previous value. Always save values you need later before overwriting.

Question 3.1 Conceptual

Operator Precedence

Evaluate: 10 + 5 * 2 - 8 / 4 % 2

Hint: Precedence order: () > *, /, % > +, -. Operations with same precedence execute left to right. Multiplication happens before addition!
Solution:
10 + 5 * 2 - 8 / 4 % 2 10 + 10 - 8 / 4 % 2 (Step 1: 5 * 2 = 10) 10 + 10 - 2 % 2 (Step 2: 8 / 4 = 2) 10 + 10 - 0 (Step 3: 2 % 2 = 0) 20 - 0 (Step 4: 10 + 10 = 20) 20 (Final Result)
