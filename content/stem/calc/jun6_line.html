Work & Line Integrals
Calc A2 Final23 June 2026
Quick reference. Work \(=\displaystyle\int_\gamma \mathbf F\cdot d\mathbf r=\int_a^b \mathbf F(\gamma(t))\cdot\gamma'(t)\,dt\). Curl test (2-D): \(\mathbf F=(P,Q)\) is conservative on a simply-connected domain iff \(P_y=Q_x\). If conservative, find a potential \(\phi\) and use the Gradient Theorem: \(\int_\gamma\mathbf F\cdot d\mathbf r=\phi(\text{end})-\phi(\text{start})\) — path-independent, no parametrisation needed.
★ Connected ProblemIs the field conservative?exam-style
Let \(\mathbf F=(2xy+y,\ x^2+x)\) and let \(\gamma\) go from \((0,0)\) to \((1,1)\).
chains: curl test · potential function · Gradient Theorem · path-independence
  1. (a)
    Apply the curl test.
    Solution\(P_y=2x+1=Q_x=2x+1\) ✓ conservative (on all of \(\mathbb R^2\), simply connected).
    Conservative.
  2. (b)
    Find a potential \(\phi\).
    Solution\(\int(2xy+y)\,dx=x^2y+xy+g(y)\); \(\partial_y=x^2+x+g'=x^2+x\Rightarrow g'=0\).
    \(\phi=x^2y+xy\).
  3. (c)
    Compute the work along \(\gamma\).
    Solution\(\phi(1,1)-\phi(0,0)=(1+1)-0=2\).
    Work \(=2\).
  4. (d)
    Would a different path from \((0,0)\) to \((1,1)\) change the answer?
    SolutionNo — conservative on a simply-connected domain ⇒ path-independent.
    Same value \(2\).
Problem 1 Direct parametrisation 10 pts
Evaluate \(\displaystyle\int_\gamma \mathbf F\cdot d\mathbf r\) for \(\mathbf F=(y,x^2)\) along \(\gamma(t)=(t,t^2)\), \(t:0\to1\).
Solution \(\gamma'(t)=(1,2t)\). \(\mathbf F(\gamma)=(t^2,t^2)\). Dot: \(t^2\cdot1+t^2\cdot2t=t^2+2t^3\). \(\int_0^1(t^2+2t^3)\,dt=\tfrac13+\tfrac12=\tfrac56\).
Work \(=\tfrac56\).
Problem 2 Curl test + potential 10 pts
\(\mathbf F=(2xy,\;x^2)\). Show it's conservative, find \(\phi\), and evaluate \(\int_\gamma\mathbf F\cdot d\mathbf r\) from \((0,0)\) to \((1,2)\) along any path.
Solution Curl test: \(P_y=2x=Q_x=2x\) ✓ conservative. Find \(\phi\): \(\int2xy\,dx=x^2y+g(y)\); \(\partial_y(x^2y+g)=x^2+g'(y)=x^2\Rightarrow g'=0\). So \(\phi=x^2y\). Gradient Theorem: \(\phi(1,2)-\phi(0,0)=2-0=2\).
\(\int_\gamma\mathbf F\cdot d\mathbf r=2\) (path-independent).
Always run the curl test first — if it passes, the potential shortcut saves the entire parametrisation.
Problem 3 Scalar line integral 10 pts
Mass of a wire: \(\displaystyle\int_\gamma (x+y)\,ds\) where \(\gamma\) is the line from \((0,0)\) to \((1,1)\). (Recall \(ds=\lVert\gamma'\rVert\,dt\).)
Solution \(\gamma(t)=(t,t)\), \(t:0\to1\), \(\gamma'=(1,1)\), \(\lVert\gamma'\rVert=\sqrt2\). Integrand \(x+y=2t\). \(\int_0^1 2t\cdot\sqrt2\,dt=\sqrt2\,[t^2]_0^1=\sqrt2\).
\(\int_\gamma(x+y)\,ds=\sqrt2\).
Scalar (arc-length) integrals use \(ds=\lVert\gamma'\rVert dt\) — don't confuse with the vector work integral, which uses \(\gamma'\,dt\) directly.
Problem 4 The vortex trap 10 pts
\(\mathbf F=\left(\frac{-y}{x^2+y^2},\frac{x}{x^2+y^2}\right)\) passes the curl test, yet \(\oint\) around the unit circle is \(2\pi\neq0\). Explain.
Solution The curl is 0 everywhere the field is defined, but the domain has a hole at the origin — it's not simply connected. The Gradient Theorem requires a simply-connected domain, so curl-free does not imply path-independent here.
Conservative ⇏ from curl alone when the domain has a hole; \(\oint=2\pi\).
This is the canonical counterexample: the hypothesis "simply connected" is doing real work.
★ ChallengeExam-level · zero curl but nonzero circulationpast-final level
Let \(\mathbf F=\Big(\dfrac{-y}{x^2+y^2},\ \dfrac{x}{x^2+y^2}\Big)\) (undefined at the origin).
chains: curl test · circulation by parametrising · the simply-connected caveat · deform the loop
  1. (a)
    Show \(\partial_xQ=\partial_yP\) everywhere except the origin.
    SolutionBoth equal \(\dfrac{y^2-x^2}{(x^2+y^2)^2}\), so the scalar curl \(\partial_xQ-\partial_yP=0\) away from \((0,0)\).
    curl \(=0\) on \(\mathbb R^2\setminus\{0\}\).
  2. (b)
    Compute \(\oint_C\mathbf F\cdot d\mathbf r\) around the unit circle (CCW).
    Solution\(x=\cos t,y=\sin t\): \(\mathbf F=(-\sin t,\cos t)\), \(d\mathbf r=(-\sin t,\cos t)\,dt\), \(\mathbf F\cdot d\mathbf r=(\sin^2t+\cos^2t)\,dt=dt\). \(\oint=\int_0^{2\pi}dt=2\pi\).
    \(2\pi\) (not \(0\)!).
  3. (c)
    Why doesn't this break "curl \(0\Rightarrow\) conservative"?
    SolutionThe domain \(\mathbb R^2\setminus\{0\}\) is not simply connected — the curve encircles the hole at the origin, so the curl-test theorem doesn't apply and \(\mathbf F\) has no global potential.
    hole in the domain.
  4. (d)
    What is the circulation around any simple closed CCW curve that (i) encloses the origin, (ii) doesn't?
    Solution(i) \(2\pi\) — deform it to the unit circle through the curl-free region (Green's on the annulus between them gives equality). (ii) \(0\) — Green's applies on the enclosed region where curl \(=0\).
    \(2\pi\) if it winds the origin, else \(0\).
