Implicit Function Theorem
Calc A2 Final23 June 2026
Quick reference. \(g(x,y)=c\) defines \(y=y(x)\) locally near a point iff \(g_y\neq0\) there. The implicit derivative is \(\dfrac{dy}{dx}=-\dfrac{g_x}{g_y}\). Where \(g_y=0\), the curve has a vertical tangent and \(y\) is not a function of \(x\). Yang's signature question: "where does the surface fail to be smooth?" → set the relevant partial (or Jacobian) to 0 and solve.

The curve is \(g(x,y)=x^3+y^3-6xy=0\).
chains: implicit FT · implicit derivative · vertical tangent · tangent line
  1. (a)
    Compute \(g_x,g_y\).
    Solution\(g_x=3x^2-6y,\ g_y=3y^2-6x\).
    as shown.
  2. (b)
    Give \(dy/dx\) by the implicit-FT formula.
    Solution\(\dfrac{dy}{dx}=-\dfrac{g_x}{g_y}=-\dfrac{x^2-2y}{y^2-2x}\).
    \(-\dfrac{x^2-2y}{y^2-2x}\).
  3. (c)
    Where does the curve have a vertical tangent?
    SolutionWhere \(g_y=0\Rightarrow y^2=2x\) (and on the curve).
    \(y^2=2x\) points.
  4. (d)
    Find the tangent line at \((3,3)\).
    Solution\(dy/dx=-\dfrac{9-6}{9-6}=-1\); line \(y-3=-(x-3)\).

Problem 1 Implicit derivative 10 pts
For each curve, find \(dy/dx\) by the implicit-FT formula.
  1. (a)
    Circle: \(x^2+y^2=25\). 3 pts
    Solution \(g=x^2+y^2-25\), \(g_x=2x\), \(g_y=2y\). \(\dfrac{dy}{dx}=-\dfrac{2x}{2y}=-\dfrac{x}{y}\).
    \(dy/dx=-x/y\), undefined at \((\pm5,0)\).
  2. (b)
    Folium: \(x^3+y^3=6xy\). 4 pts
    Solution \(g=x^3+y^3-6xy\), \(g_x=3x^2-6y\), \(g_y=3y^2-6x\). \(\dfrac{dy}{dx}=-\dfrac{3x^2-6y}{3y^2-6x}=-\dfrac{x^2-2y}{y^2-2x}\).
    \(dy/dx=-\dfrac{x^2-2y}{y^2-2x}\).
  3. (c)
    \(e^{xy}+x=y\). 3 pts
    Solution \(g=e^{xy}+x-y\), \(g_x=ye^{xy}+1\), \(g_y=xe^{xy}-1\). \(\dfrac{dy}{dx}=-\dfrac{ye^{xy}+1}{xe^{xy}-1}\).
    \(dy/dx=-\dfrac{ye^{xy}+1}{xe^{xy}-1}\).
Problem 2 Where does it fail to be smooth? 10 pts
The curve \(y^2=x^3-x\) (an elliptic curve). Find every point where \(y\) fails to be a smooth function of \(x\).
Solution \(g=y^2-x^3+x\), \(g_y=2y\). IFT fails where \(g_y=0\Rightarrow y=0\). On the curve, \(y=0\Rightarrow x^3-x=0\Rightarrow x=0,\pm1\).
Non-smooth (vertical-tangent) points: \((0,0),(1,0),(-1,0)\).
At these points the tangent is vertical, so \(x\) is locally a function of \(y\) instead — swap roles when \(g_y=0\) but \(g_x\neq0\).
Problem 3 Solve for a parameter 10 pts
For \(g(x,y)=x^2+\alpha y^2-1=0\), find the value(s) of \(\alpha\) and the point where the curve has a single horizontal-tangent failure of "\(x\) as a function of \(y\)" at \((0,1)\).
Solution "\(x\) as a function of \(y\)" needs \(g_x\neq0\); it fails where \(g_x=2x=0\Rightarrow x=0\). At \((0,1)\) the constraint gives \(\alpha(1)^2=1\Rightarrow\alpha=1\).
\(\alpha=1\); failure point \((0,1)\) (and \((0,-1)\)).
The recipe: set the relevant partial to 0, then use the equation \(g=0\) to pin the constant. This is exactly the 2025-final style.
