Quick reference. Two types: infinite limits \(\displaystyle\int_a^\infty f=\lim_{b\to\infty}\int_a^b f\), and unbounded integrand (singularity at \(c\)) \(\displaystyle\int_a^c f=\lim_{t\to c^-}\int_a^t f\). The integral converges iff the limit is finite. Useful facts: \(\int_1^\infty x^{-p}\,dx\) converges iff \(p>1\); \(\int_0^1 x^{-p}\,dx\) converges iff \(p<1\). For sign-changing or hard integrands, use the comparison test: if \(0\le f\le g\) and \(\int g\) converges, so does \(\int f\).
Problem 1 Evaluate or Determine Convergence 10 pts
For each integral below, state whether it converges; if it does, evaluate it exactly.
  1. (a)
    \(\displaystyle\int_1^{\infty}\frac{1}{x^2}\,dx\) 2 pts
    Solution \(\int_1^b x^{-2}dx=\big[-x^{-1}\big]_1^b=1-\tfrac1b\to1\) as \(b\to\infty\). (Here \(p=2>1\), so it converges.)
    Converges to \(1\).
  2. (b)
    \(\displaystyle\int_1^{\infty}\frac{1}{\sqrt{x}}\,dx\) 2 pts
    Solution \(\int_1^b x^{-1/2}dx=\big[2\sqrt{x}\big]_1^b=2\sqrt{b}-2\to\infty\). (Here \(p=\tfrac12\le1\).)
    Diverges.
  3. (c)
    \(\displaystyle\int_0^{1}\frac{1}{\sqrt{x}}\,dx\) (singularity at \(x=0\)) 2 pts
    Solution \(\int_t^1 x^{-1/2}dx=\big[2\sqrt{x}\big]_t^1=2-2\sqrt{t}\to2\) as \(t\to0^+\). (Here \(p=\tfrac12<1\), so it converges.)
    Converges to \(2\).
    Contrast with (b): the same integrand converges near 0 but diverges at ∞ — the \(p\)-thresholds are opposite.
  4. (d)
    \(\displaystyle\int_0^{\infty} e^{-2x}\,dx\) 2 pts
    Solution \(\int_0^b e^{-2x}dx=\big[-\tfrac12 e^{-2x}\big]_0^b=\tfrac12-\tfrac12 e^{-2b}\to\tfrac12\).
    Converges to \(\tfrac12\).
  5. (e)
    \(\displaystyle\int_{-\infty}^{\infty}\frac{1}{1+x^2}\,dx\) 2 pts
    Solution Split at 0: \(\int_{-\infty}^{\infty}\frac{dx}{1+x^2}=\big[\arctan x\big]_{-\infty}^{\infty}=\tfrac{\pi}{2}-(-\tfrac{\pi}{2})=\pi\).
    Converges to \(\pi\).
    A two-sided improper integral must converge on both halves independently — here each half gives \(\tfrac{\pi}{2}\).
Problem 2 Radioactive Decay Lab — Total Quantities 10 pts
A physics lab measures decaying signals over unbounded time. Each part below is the "total accumulated" quantity — an improper integral. Determine convergence and evaluate where possible.
  1. (a)
    Total emission: \(\displaystyle\int_0^{\infty} x\,e^{-x}\,dx\). 2 pts
    Solution Integration by parts: \(\int x e^{-x}dx=-x e^{-x}-e^{-x}\). Evaluate \(0\to b\): \(\big[-(x+1)e^{-x}\big]_0^b=1-(b+1)e^{-b}\to1\) (since \((b+1)e^{-b}\to0\)).
    Converges to \(1\).
  2. (b)
    Signal energy: \(\displaystyle\int_1^{\infty}\frac{\ln x}{x}\,dx\). 2 pts
    Solution Let \(u=\ln x\): \(\int\frac{\ln x}{x}dx=\tfrac12(\ln x)^2\). Evaluate \(1\to b\): \(\tfrac12(\ln b)^2-0\to\infty\).
    Diverges.
  3. (c)
    Decay tail: \(\displaystyle\int_1^{\infty}\frac{1}{x(x+1)}\,dx\). 2 pts
    Solution Partial fractions: \(\frac{1}{x(x+1)}=\frac1x-\frac1{x+1}\). Antiderivative \(\ln\frac{x}{x+1}\). Evaluate \(1\to b\): \(\ln\frac{b}{b+1}-\ln\frac12\to\ln1-\ln\tfrac12=\ln2\).
    Converges to \(\ln 2\).
  4. (d)
    Bounded estimate: does \(\displaystyle\int_1^{\infty}\frac{\sin^2 x}{x^2}\,dx\) converge? (Use comparison.) 2 pts
    Solution Since \(0\le \sin^2 x\le1\), we have \(0\le\frac{\sin^2 x}{x^2}\le\frac{1}{x^2}\). And \(\int_1^\infty x^{-2}dx=1\) converges. By comparison, the integral converges.
    Converges (by comparison with \(1/x^2\)). Exact value not required.
    Comparison proves convergence without finding the value — the key skill when no elementary antiderivative exists.
  5. (e)
    Singularity inside: \(\displaystyle\int_0^{2}\frac{1}{(x-1)^2}\,dx\). (Careful — singularity at \(x=1\).) 2 pts
    Solution The integrand blows up at \(x=1\), inside \([0,2]\). Split: \(\int_0^1+\int_1^2\). For \(\int_t^... \) near 1, \(\int\frac{dx}{(x-1)^2}=-\frac{1}{x-1}\). On \([1,2]\): \(\lim_{t\to1^+}\big[-\tfrac{1}{x-1}\big]_t^2=-1+\lim_{t\to1^+}\tfrac{1}{t-1}=+\infty\).
    Diverges (the singularity at \(x=1\) makes it blow up).
    Trap: integrating blindly from 0 to 2 gives a finite "answer" of \(-2\), which is wrong — you must detect the interior singularity first.
