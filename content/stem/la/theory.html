Chapter 3 · Theory

Gaussian Elimination & Pre-image

EZ pp. 60–95pre-image · RREF
Big idea. Every linear system \(A\mathbf x=\mathbf b\) is a pre-image problem. Gaussian elimination is a sequence of solution-preserving row operations that brings the augmented matrix to a unique RREF — from which existence, uniqueness, and the general solution can be read off immediately.
Definition Pre-image problem
Given a linear map \(L\) and a target \(\mathbf b\), find every \(\mathbf x\) with \(L(\mathbf x)=\mathbf b\). In matrix form: solve \(A\mathbf x=\mathbf b\). The set of solutions is either empty, a single point, or an affine subspace (a translate of \(\operatorname{Ker}(L)\) — the homogeneous solution).
Definition "Jectivity" — injective, surjective, bijective
Injective: distinct inputs go to distinct outputs (\(L(\mathbf x)=L(\mathbf y)\Rightarrow\mathbf x=\mathbf y\)). Surjective: every target is achievable (\(\operatorname{Ran}(L)=W\)). Bijective: both. For linear maps, injective \(\iff\operatorname{Ker}(L)=\{0\}\).
Theorem Three elementary row operations preserve solutions
Swap, scale by \(c\ne 0\), and \(R_i\to R_i+cR_j\) all give equivalent systems. Each is invertible (swap is self-inverse; the others undo by negating the multiplier), so the solution set is unchanged. This is what licenses the whole Gaussian-elimination machinery.
Definition Reduced Row Echelon Form (RREF)
Four conditions: (R1) zero rows at the bottom; (R2) each pivot is \(1\); (R3) pivots step strictly to the right going down; (R4) each pivot is the only nonzero entry in its column. Echelon form (REF) is (R1)–(R3); reducing further to (R4) makes it reduced.
Theorem Uniqueness of RREF (the central result of Ch.3)
Every matrix \(A\) has a unique RREF — independent of which row operations you use to get there. Proof idea. The pivot columns are exactly those columns of \(A\) that are linearly independent from all earlier columns — an intrinsic property of \(A\). For each non-pivot column, its RREF entries are the unique coefficients expressing it as a combination of earlier pivot columns — also intrinsic. So both pivot pattern and entries are determined.

Algebra of Linear Maps

EZ pp. 96–140matrices · LU
Big idea. Function composition is matrix multiplication. Gaussian elimination secretly factors \(A=LU\) (or \(PA=LU\)): the multipliers you use during elimination fill the lower triangle of \(L\), and the resulting echelon matrix is \(U\). This bridges the algorithmic view (Ch.3) and the algebraic view.
Theorem Matrix mult \(=\) function composition
If \(L_A:\mathbb R^n\to\mathbb R^m\) and \(L_B:\mathbb R^p\to\mathbb R^n\) are linear maps with matrices \(A\) and \(B\), then \(L_A\circ L_B\) has matrix \(AB\). The \((i,j)\) entry of \(AB\) is the dot product of row \(i\) of \(A\) with column \(j\) of \(B\).
Definition Elementary matrices & block matrices
An elementary matrix performs one row operation by left-multiplication. Block matrices partition rows/columns so that products factor block-by-block, identical-looking formulas to scalar matrices — provided the blocks have compatible sizes. This is the key trick behind LU.
Theorem LU existence (when no pivoting needed)
If Gaussian elimination on \(A\) never needs a row swap, then \(A=LU\) where \(L\) is unit lower triangular (the multipliers) and \(U\) is the resulting upper triangular matrix. Equivalent condition: every leading principal minor \(\det(A^{(k)})\) is nonzero.
Theorem PLU decomposition (general case)
For any square \(A\), there exists a permutation \(P\) such that \(PA=LU\). \(P\) records the row swaps performed during elimination; \(L,U\) are as before but built from the pivoted rows.
Theorem Uniqueness of LU (with diagonal normalisation)
If \(A=LU\) where \(L\) is unit lower triangular and \(U\) is upper triangular, the factorisation is unique. Proof: \(L_1U_1=L_2U_2\Rightarrow L_2^{-1}L_1=U_2U_1^{-1}\). The left side is unit lower triangular and the right is upper triangular — only the identity matrix is both. So \(L_1=L_2\) and \(U_1=U_2\). \(\blacksquare\)
Chapter 5 · Theory

Abstract Vector Spaces

EZ pp. 141–180basis · rank-nullity
Big idea. Linear algebra is not about \(\mathbb R^n\) specifically — polynomial spaces, matrix spaces, and function spaces obey the same rules. A choice of basis gives an isomorphism (the coordinate map) to \(\mathbb R^{\dim V}\), turning every abstract problem into a concrete one in \(\mathbb R^n\).
Definition Vector space axioms (the 8 rules)
A set \(V\) with \(+\) and scalar multiplication is a vector space if: (1) \(+\) commutative, (2) \(+\) associative, (3) \(0\) exists, (4) inverses \(-v\), (5) \(1\cdot v=v\), (6) \((ab)v=a(bv)\), (7) \(a(u+v)=au+av\), (8) \((a+b)v=av+bv\). Examples: \(\mathbb R^n\), \(P_n\), \(M_{m\times n}\), \(C[a,b]\) (continuous functions on \([a,b]\)).
Definition Linear independence, spanning, basis, dimension
\(\{v_i\}\) is independent if \(\sum c_iv_i=0\Rightarrow\) all \(c_i=0\); spanning if every \(v\in V\) is \(\sum c_iv_i\); a basis is both. Dimension = the common cardinality of every basis (unique by the dimension theorem).
Theorem Dimension is well-defined
Any two bases of a finite-dimensional vector space have the same number of elements. Proof idea (Steinitz exchange). Given bases \(B,C\), swap out elements of \(B\) for elements of \(C\) one at a time; independence forces \(|C|\le|B|\), and symmetrically \(|B|\le|C|\).
Theorem Coordinate map is an isomorphism
For a basis \(\mathcal B\) of \(V\) (\(\dim V=n\)), the map \([\,\cdot\,]_{\mathcal B}:V\to\mathbb R^n\) sending \(v\mapsto\) its coordinate vector is a bijective linear map. So every \(n\)-dim space is \emph{isomorphic} to \(\mathbb R^n\); abstract problems reduce to concrete ones.
Theorem Change of basis formula
\([v]_{\mathcal C}=I_{\mathcal C\leftarrow\mathcal B}\,[v]_{\mathcal B}\) where \(I_{\mathcal C\leftarrow\mathcal B}\) has the \(\mathcal B\)-vectors written in \(\mathcal C\)-coordinates as its columns. For a linear map \(T:V\to V\): \([T]_{\mathcal C}=I_{\mathcal C\leftarrow\mathcal B}\,[T]_{\mathcal B}\,I_{\mathcal B\leftarrow\mathcal C}\) (similarity transformation).
Theorem Rank-Nullity
For linear \(L:V\to W\) with \(\dim V<\infty\): \(\dim V=\operatorname{rank}(L)+\text{nullity}(L)\). Proof. Take a basis of \(\operatorname{Ker}(L)\) and extend to a basis of \(V\). The images of the extension vectors form a basis of \(\operatorname{Ran}(L)\): they span (because the kernel vectors vanish) and are independent (any combination summing to 0 lifts to a kernel vector). \(\blacksquare\)
Theorem Row rank \(=\) column rank
For any matrix \(A\), \(\dim\operatorname{Row}(A)=\dim\operatorname{Col}(A)=\operatorname{rank}(A)\). Both count the number of pivots in RREF — proved via the pivot column characterisation from Ch.3.
Criterion Injective ↔ surjective for square matrices
If \(L:V\to V\) with \(\dim V<\infty\): injective \(\iff\) surjective \(\iff\) invertible. Direct corollary of rank-nullity. In matrix terms: \(\operatorname{rank}(A)=n\iff\) full rank \(\iff\) \(A\) invertible.
Theorem Dimension formula for subspace algebra
For subspaces \(U,W\subseteq V\): \(\dim(U+W)=\dim U+\dim W-\dim(U\cap W)\). Special case: \(U\oplus W\) (direct sum) means \(U\cap W=\{0\}\), so \(\dim(U+W)=\dim U+\dim W\).

Inner Product Spaces

EZ pp. 181–230orthogonality · projections
Big idea. An inner product is a way to measure angles and lengths inside an abstract vector space. Every finite-dim inner product space is isomorphic to \(\mathbb R^n\) with the dot product (via Cholesky) — so orthonormal bases let us pretend we're always in Euclidean space.
Definition Inner product (4 axioms)
\(\langle\cdot,\cdot\rangle:V\times V\to\mathbb R\) satisfying: (1) symmetric \(\langle u,v\rangle=\langle v,u\rangle\); (2) linear in first slot \(\langle au+bw,v\rangle=a\langle u,v\rangle+b\langle w,v\rangle\); (3) positive \(\langle v,v\rangle\ge0\); (4) definite \(\langle v,v\rangle=0\iff v=0\). Induces norm \(\|v\|=\sqrt{\langle v,v\rangle}\).
Theorem Cauchy–Schwarz inequality
\(|\langle u,v\rangle|\le\|u\|\cdot\|v\|\), with equality iff \(u,v\) collinear. Proof. The polynomial \(p(t)=\|u-tv\|^2=\|u\|^2-2t\langle u,v\rangle+t^2\|v\|^2\) is \(\ge 0\) for all \(t\), so its discriminant \(4\langle u,v\rangle^2-4\|u\|^2\|v\|^2\le0\). \(\blacksquare\)
Theorem Fundamental Theorem of Linear Algebra (FTLA)
For any matrix \(A\): \(\operatorname{Row}(A)\perp\operatorname{Null}(A)\) and \(\operatorname{Col}(A)\perp\operatorname{LeftNull}(A)\), and these give orthogonal direct-sum decompositions: \(\mathbb R^n=\operatorname{Row}(A)\oplus\operatorname{Null}(A)\), \(\mathbb R^m=\operatorname{Col}(A)\oplus\operatorname{LeftNull}(A)\). Dimensions: \(\dim\operatorname{Row}=\dim\operatorname{Col}=r\); \(\dim\operatorname{Null}=n-r\); \(\dim\operatorname{LeftNull}=m-r\).
Definition Gram matrix & positive definiteness
For columns \(\mathbf a_1,\dots,\mathbf a_n\) of \(A\), the Gram matrix is \(G=A^TA\) with \(G_{ij}=\mathbf a_i\cdot\mathbf a_j\). \(G\) is always symmetric positive semi-definite; positive definite iff the \(\mathbf a_i\) are linearly independent (\(A\) has independent columns).
Theorem Cholesky decomposition
Every positive-definite matrix \(G\) factors uniquely as \(G=LL^T\) where \(L\) is invertible lower triangular with positive diagonal entries. Why it matters: any finite-dim inner-product space is isomorphic (via change of basis) to Euclidean space with the dot product. Equivalently, every positive-definite \(G\) is the Gram matrix of some basis.
Definition Orthonormal basis (ONB) & orthogonal matrix
An ONB \(q_1,\dots,q_n\) has \(\|q_i\|=1\) and \(q_i\perp q_j\) for \(i\neq j\). An orthogonal matrix \(Q\) is one whose columns form an ONB; equivalently, \(Q^TQ=I\), i.e., \(Q^{-1}=Q^T\). Orthogonal matrices preserve inner products: \(\langle Q\mathbf x,Q\mathbf y\rangle=\langle\mathbf x,\mathbf y\rangle\) — they are isometries.
Theorem Gram-Schmidt → QR decomposition
Every matrix \(A\) with independent columns factors as \(A=QR\), with \(Q\) orthogonal (its columns are the Gram-Schmidt orthonormalised columns of \(A\)) and \(R\) upper triangular with positive diagonal. Three proof perspectives in EZ: row-by-row construction; column-shearing producing \(R\); and the algebraic \(R=Q^TA\) once \(Q\) is known.
Theorem Orthogonal projection theorem & least squares
For a subspace \(U\subseteq V\), every \(v\) decomposes uniquely as \(v=v_U+v_{U^\perp}\) with \(v_U\in U\) minimising \(\|v-u\|\) over all \(u\in U\). Matrix form: project onto \(\operatorname{Col}(A)\) using \(P=A(A^TA)^{-1}A^T\). The least-squares solution to (possibly inconsistent) \(A\mathbf x=\mathbf b\) is the exact solution to \(A^TA\mathbf x=A^T\mathbf b\) — the normal equations.
Chapter 7 · Theory

Determinants

EZ pp. 231–270multilinear · volume
Big idea. The determinant is the unique multilinear, alternating function of the columns of a square matrix that equals \(1\) on the identity. Geometrically, it's the signed volume of the parallelepiped spanned by the columns.
Definition Determinant by three axioms
\(\det:M_{n\times n}\to\mathbb R\) satisfying: (1) multilinear in each column; (2) alternating: \(\det=0\) if two columns are equal (equivalently: sign flips on column swap); (3) normalised: \(\det(I)=1\). These three uniquely determine \(\det\).
Theorem Existence: permutation formula
\(\det(A)=\sum_{\sigma\in S_n}\operatorname{sgn}(\sigma)\,a_{1,\sigma(1)}\cdots a_{n,\sigma(n)}\) — Leibniz formula. Sum over all \(n!\) permutations; \(\operatorname{sgn}\) is \(+1\) for even, \(-1\) for odd. Proves existence; satisfies the three axioms.
Theorem Multiplicativity
\(\det(AB)=\det(A)\det(B)\) for square \(A,B\). Consequence: \(A\) is invertible iff \(\det(A)\neq0\), and \(\det(A^{-1})=1/\det(A)\). Also \(\det(A^T)=\det(A)\).
Theorem Volume interpretation
\(|\det(A)|\) is the \(n\)-dimensional volume of the parallelepiped spanned by the columns of \(A\). Sign tells you orientation (positive = right-handed). This is why row operations rescale or sign-flip the determinant: scaling a row by \(c\) scales \(\det\) by \(c\); a row swap flips the sign; adding a multiple of one row to another doesn't change \(\det\) (the parallelepiped has the same base and height).
Theorem Cofactor expansion
Pick any row (or column) \(i\): \(\det(A)=\sum_j (-1)^{i+j}\,a_{ij}\,M_{ij}\), where \(M_{ij}\) is the \((n-1)\times(n-1)\) minor (delete row \(i\), col \(j\)). Best when a row/column has many zeros. Specialises to base-times-height interpretation in 2D, scalar triple product in 3D.

Eigenstuff, Spectral Thm, SVD

EZ pp. 271–349the final-exam centerpiece
Big idea. An eigenbasis turns a linear map into a diagonal matrix. Not every matrix has one — but every matrix admits at least a Schur form, every symmetric matrix admits a clean spectral decomposition with an orthonormal eigenbasis, and every matrix (even rectangular) admits an SVD.
Definition Eigenvalue, eigenvector, eigenspace
\(\lambda\in\mathbb C\) is an eigenvalue of \(A\) if \(A\mathbf v=\lambda\mathbf v\) for some \(\mathbf v\ne0\) (an eigenvector). The set of eigenvectors for a fixed \(\lambda\) plus \(\mathbf 0\) is the eigenspace \(E_\lambda=\operatorname{Ker}(A-\lambda I)\).
Theorem Characteristic polynomial
Eigenvalues are exactly the roots of \(p_A(\lambda)=\det(\lambda I-A)\). For \(n\times n\) matrices, \(p_A\) has degree \(n\); by FTA it has \(n\) roots over \(\mathbb C\) (counting multiplicity). Real matrices can have complex-conjugate eigenvalue pairs.
Definition Algebraic vs geometric multiplicity
Algebraic multiplicity of \(\lambda\) = its multiplicity as a root of \(p_A\). Geometric multiplicity = \(\dim E_\lambda\). Always \(1\le\) geometric \(\le\) algebraic. Equality (for all \(\lambda\)) iff \(A\) is diagonalisable.
Theorem Diagonalisability
\(A\) is diagonalisable iff it has \(n\) linearly independent eigenvectors. Then \(A=PDP^{-1}\) with \(D\) diagonal of eigenvalues and \(P\) whose columns are eigenvectors. Sufficient condition: \(n\) distinct eigenvalues (eigenvectors for distinct eigenvalues are automatically independent).
Theorem Schur decomposition
Every square matrix \(A\) (over \(\mathbb C\)) admits \(A=QTQ^*\) where \(Q\) is unitary and \(T\) is upper triangular. The diagonal of \(T\) contains the eigenvalues. Real-Schur form: over \(\mathbb R\), \(T\) is block upper triangular with \(1\times1\) (real eigenvalues) and \(2\times2\) (conjugate pairs) blocks.
Theorem Spectral Theorem (symmetric matrices)
If \(A=A^T\) (real symmetric), then \(A=Q\Lambda Q^T\) for an orthogonal \(Q\) and diagonal \(\Lambda\). Real symmetric matrices have: all real eigenvalues; an orthonormal eigenbasis; and the spectral decomposition exists for every symmetric matrix (no diagonalisability obstruction).
Theorem Singular Value Decomposition (SVD)
For any \(m\times n\) matrix \(A\) (square or not), \(A=U\Sigma V^T\) with \(U\in O(m)\), \(V\in O(n)\), and \(\Sigma\) diagonal-rectangular with non-negative singular values \(\sigma_1\ge\sigma_2\ge\cdots\ge\sigma_r>0\), where \(r=\operatorname{rank}(A)\). The \(\sigma_i^2\) are the nonzero eigenvalues of \(A^TA\) (which is symmetric PSD), \(V\) holds the eigenvectors of \(A^TA\), and \(U\) holds those of \(AA^T\).
Theorem Eckart-Young: best low-rank approximation
Given \(A=U\Sigma V^T\), the rank-\(k\) matrix \(A_k=\sum_{i=1}^k\sigma_iu_iv_i^T\) (truncated SVD) is the closest rank-\(k\) matrix to \(A\) in both Frobenius and operator norms. This is the foundation of PCA, image compression, and noise reduction.
Theorem Classification of quadratic forms
A quadratic form \(q(\mathbf x)=\mathbf x^TA\mathbf x\) (with \(A\) symmetric) is classified by the signs of \(A\)'s eigenvalues: all positive → positive-definite (ellipsoid); all negative → negative-definite; mixed → indefinite (hyperboloid/saddle); some zero → degenerate (paraboloid/cylinder). Sylvester's law of inertia: this signature is invariant under any change of basis.
