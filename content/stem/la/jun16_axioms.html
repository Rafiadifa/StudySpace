Vector Space Axioms
Lin Alg Final25 June 2026
⌁ Write the 8 axioms from memory
A vector space is a set with addition and scalar multiplication satisfying 8 axioms. List all 8, then reveal.
The 8 axiomsVector space
SolutionAddition: (1) commutative, (2) associative, (3) zero vector \(\mathbf 0\), (4) additive inverses. Scalar mult: (5) \(1\mathbf v=\mathbf v\), (6) \(a(b\mathbf v)=(ab)\mathbf v\), (7) \(a(\mathbf u+\mathbf v)=a\mathbf u+a\mathbf v\), (8) \((a+b)\mathbf v=a\mathbf v+b\mathbf v\).
Test itIs it a vector space?
Are polynomials of degree exactly 2 a vector space?
SolutionNo — not closed under addition: \((x^2)+(-x^2+x)=x\) has degree 1. Fails the closure needed for axiom structure. (Degree \(\le2\) is a vector space.)
