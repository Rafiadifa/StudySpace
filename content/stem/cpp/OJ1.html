Problem 1 Christmas Tree Nested loops
Read an integer \(n\). Print a triangle of * with \(n\) rows: row \(i\) has \((n-i)\) leading spaces followed by \((2i-1)\) stars. After the tree, print a trunk: \((n-1)\) spaces then a single |.
Input
3
Output
  *
 ***
*****
  |
Approach Two stacked loops. The outer loop runs rows \(i=1\ldots n\). Inside, print \((n-i)\) spaces to centre the row, then \((2i-1)\) stars, then a newline. After the loop, print the trunk separately. 
// each row: (n-i) spaces, then (2i-1) stars
for (int i = 1; i <= n; i++) {
    for (int j = 1; j <= n - i; j++) cout << " ";
    for (int k = 1; k <= 2*i - 1; k++) cout << "*";
    cout << endl;
}
// trunk
for (int j = 1; j <= n - 1; j++) cout << " ";
cout << "|" << endl;
Watch the off-by-one: row \(i\) has \(2i-1\) stars (odd counts 1,3,5,…), and centring needs exactly \(n-i\) leading spaces.
Problem 2 Weighted GPA Lookup table
Read \(n\) courses. Each course has a credit \(c\) and a score \(s\). Convert each score to a grade point, then output the credit-weighted GPA \(=\dfrac{\sum c\cdot gp}{\sum c}\), printed to 2 decimal places.

Grade scale: \(\ge90\to4.0\), \(\ge85\to3.6\), \(\ge80\to3.3\), \(\ge77\to3.0\), \(\ge73\to2.6\), \(\ge70\to2.3\), \(\ge67\to2.0\), \(\ge63\to1.6\), \(\ge60\to1.3\), else \(0.0\).
Input
2
3 92
4 75
Output
3.13
Approach Put the score→grade-point mapping in a helper function (a descending if-ladder). Accumulate totalWeightedGP += c*gp and totalCredits += c, then divide. Use fixed + setprecision(2) for the output format. 
double getGradePoint(int s) {
    if (s >= 90) return 4.0;
    else if (s >= 85) return 3.6;
    else if (s >= 80) return 3.3;
    else if (s >= 77) return 3.0;
    else if (s >= 73) return 2.6;
    else if (s >= 70) return 2.3;
    else if (s >= 67) return 2.0;
    else if (s >= 63) return 1.6;
    else if (s >= 60) return 1.3;
    else return 0.0;
}
// in main: accumulate then divide
totalWeightedGP += c * getGradePoint(s);
totalCredits += c;
cout << fixed << setprecision(2) << totalWeightedGP / totalCredits;
Guard against totalCredits == 0 before dividing. Keep the GPA in double — integer division would silently truncate.
Problem 3 Goldbach Pair Primality
Read an even integer \(n\). Find the pair of primes \(p+q=n\) with the smallest \(p\) (so \(p\le q\)), and print "p q".
Input
28
Output
5 23
Approach Loop \(p\) from 2 up to \(n/2\). For each \(p\), set \(q=n-p\); if both are prime, print and stop. The first hit is automatically the smallest \(p\). Primality by trial division up to \(\sqrt{\text{num}}\). 
bool isPrime(int num) {
    if (num < 2) return false;
    for (int i = 2; i <= sqrt(num); i++)
        if (num % i == 0) return false;
    return true;
}
for (int p = 2; p <= n/2; p++) {
    int q = n - p;
    if (isPrime(p) && isPrime(q)) { cout << p << " " << q; break; }
}
Stopping \(p\) at \(n/2\) avoids printing the mirror pair twice and guarantees \(p\le q\).
Problem 4 Best Time to Buy & Sell Stock Single scan
Read \(n\) daily prices. You may buy once and sell once later. Print the maximum profit (or 0 if no profitable trade exists).
Input
6
7 1 5 3 6 4
Output
5
Approach One pass. Track the minimum price seen so far. At each day, the best sale today is price - minPrice; keep the running maximum of that. \(O(n)\) time, \(O(1)\) memory — no need to store the array. 
int minPrice = INT_MAX, maxProfit = 0;
for (int i = 0; i < n; i++) {
    int price; cin >> price;
    if (price < minPrice) minPrice = price;
    int profit = price - minPrice;
    if (profit > maxProfit) maxProfit = profit;
}
cout << maxProfit;
Buy-then-sell ordering is automatic: minPrice only ever holds a price from a day at or before the current one, so the profit is always "sell after buy".
