Section A · Fundamentals (Q1–Q4)
Q1Output prediction5 pts
What prints, and why? 
int a = 7, b = 2;
cout << a / b << " " << a % b << " " << (double)a / b;
Answer 3 1 3.5. Integer division truncates (7/2=3), modulo gives the remainder (1), and casting one operand to double forces floating-point division (3.5).
Q2Loop & control flow10 pts
Write a function int countVowels(const string&) that returns the number of vowels (case-insensitive).
Answer 
int countVowels(const string& s) {
    int n = 0;
    for (char c : s) {
        char l = tolower(c);
        if (l=='a'||l=='e'||l=='i'||l=='o'||l=='u') n++;
    }
    return n;
}
Q3Pass by value vs reference8 pts
Write swap two ways — one that works, one that doesn't — and explain the difference.
Answer 
void badSwap(int a, int b)   { int t=a; a=b; b=t; }   // copies — no effect
void goodSwap(int& a, int& b) { int t=a; a=b; b=t; }   // aliases — works
By value, the function gets copies; changes die at return. By reference, the parameters alias the caller's variables, so the swap persists.
Q4Pointers & arrays7 pts
After int arr[]={10,20,30}; int* p=arr; what are *(p+2) and *p+2?
Answer *(p+2) = 30 (move pointer two ints forward, then dereference). *p+2 = 12 (dereference first → 10, then add 2). Precedence: unary * binds tighter than binary +.
Section B · Classes & OOP (Q5–Q8)
Q5Constructor & encapsulation10 pts
Design a Temperature class storing Celsius privately, with a constructor, toFahrenheit(), and a setter that rejects values below −273.15.
Answer 
class Temperature {
    double celsius;
public:
    Temperature(double c = 0) { setCelsius(c); }
    void setCelsius(double c) { if (c >= -273.15) celsius = c; }
    double toFahrenheit() const { return celsius * 9.0/5.0 + 32; }
};
The constructor reuses the validating setter — one place owns the invariant.

Q7Operator overloading10 pts
For a Fraction{num, den}, overload operator* and operator<<. (No need to reduce.)
Answer 
Fraction operator*(const Fraction& r) const {
    return Fraction(num * r.num, den * r.den);
}
friend ostream& operator<<(ostream& os, const Fraction& f) {
    return os << f.num << "/" << f.den;
}
Q8Find the bug8 pts
Why does this leak / misbehave? 
class Base { public: ~Base(){} };
class Derived : public Base { int* buf = new int[100]; public: ~Derived(){ delete[] buf; } };
Base* b = new Derived(); delete b;
Answer ~Base isn't virtual, so delete b only runs ~Base — ~Derived never fires and buf leaks. Fix: virtual ~Base(){}. Any class meant to be deleted through a base pointer needs a virtual destructor.
Section C · Applied (Q9–Q10)
Q9friend function10 pts
Write a non-member bool heavier(const Box&, const Box&) that compares a private weight. Why must it be a friend?
Answer 
class Box {
    double weight;
public:
    Box(double w) : weight(w) {}
    friend bool heavier(const Box& a, const Box& b);
};
bool heavier(const Box& a, const Box& b) { return a.weight > b.weight; }
It's not a member but needs the private weight, so friend grants it access.
Q10Exception handling10 pts
Write a safeDivide(int a, int b) that throws on b == 0, and a caller that reports the error without crashing.
Answer 
int safeDivide(int a, int b) {
    if (b == 0) throw runtime_error("divide by zero");
    return a / b;
}
try { cout << safeDivide(10, 0); }
catch (const exception& e) { cout << e.what(); }
