Problem 1 Number Records & betterRecord friend function
Each record holds an index and a value. Its score is: digit-sum of value, \(+100\) if the value is prime, \(+50\) if it is a palindrome. Read \(n\) values (1-indexed) and \(q\) queries. Each query gives two indices \(a,b\); print the index and score of the "better" record. Better = higher score; tie-break by smaller value; then smaller index.
Input
3 1
13 22 7
1 3
Output
3 107
Approach Three free helpers — getDigitSum, isPrime, isPalindrome. The class exposes getScore(); the comparison betterRecord is declared a friend so it can read the private value and index directly for the tie-breaks. 
class NumberRecord {
    int index, value;
public:
    NumberRecord(int i, int v) : index(i), value(v) {}
    int getScore() const {
        int s = getDigitSum(value);
        if (isPrime(value))      s += 100;
        if (isPalindrome(value)) s += 50;
        return s;
    }
    friend bool betterRecord(const NumberRecord&, const NumberRecord&);
};

bool betterRecord(const NumberRecord& a, const NumberRecord& b) {
    if (a.getScore() != b.getScore()) return a.getScore() > b.getScore();
    if (a.value != b.value)           return a.value < b.value;
    return a.index < b.index;
}
The friend keyword is the whole point: betterRecord is not a member, yet touches a.value and a.index. Push a dummy record at index 0 so the input's 1-indexing lines up with the vector.
Problem 2 Sort Shapes by Area abstract base
Read \(n\) shapes. Each line is name type ...dims: R w h (rectangle), T b h (triangle, area \(=\tfrac12 bh\)), C r (circle, area \(=\pi r^2\)). Sort ascending by area and print each name area to 2 decimals.
Input
3
A C 2
B R 3 4
D T 5 2
Output
D 5.00
B 12.00
A 12.57
Approach Abstract Shape with pure virtual area(). Rectangle, Triangle, Circle each override it. Store Shape* pointers, sort by area() (the call dispatches to the right override), then print. 
class Shape {
protected: string id;
public:
    Shape(const string& id) : id(id) {}
    virtual ~Shape() {}
    string name() const { return id; }
    virtual double area() const = 0;   // pure virtual
};
class Circle : public Shape {
    double r;
public:
    Circle(const string& id, double r) : Shape(id), r(r) {}
    double area() const override { return 3.141592653589793 * r * r; }
};
// sort ascending by polymorphic area, then print
sort(shapes.begin(), shapes.end(),
     [](Shape* a, Shape* b){ return a->area() < b->area(); });
cout << fixed << setprecision(2);
Sorting Shape* and calling a->area() in the comparator is the polymorphism payoff — one comparator, three shape formulas. Remember to delete every new'd shape at the end.
